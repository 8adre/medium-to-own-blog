# Medium to own blog

_Switch from Medium to your own blog in a few minutes._

## Usage

_Requires [Node.js](https://nodejs.org/en/)_

```bash
npx medium-to-own-blog
```

## Technical stack

- [Gatsby](https://www.gatsbyjs.org/)
- [GitHub](https://github.com)
- [Netlify](https://netlify.com)
